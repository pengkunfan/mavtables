Detailed Description

A thin wrapper around Unix system calls.

The purpose of this is to allow system calls to be mocked during testing.

See the following man pages for documentation:

  • man 2 bind
  • man 2 close
  • man 2 socket
  • man 2 ioctl
  • man 7 ip
  • man 2 open
  • man 2 poll
  • man 2 read
  • man 2 recvfrom
  • man 2 sendto
  • man 2 termios
  • man 2 write
