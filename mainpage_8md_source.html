7 ## Introduction
8 
9 A MAVLink router and firewall. It can connect 2 or more MAVLink endpoints such
10 as autopilots, ground control software, loggers, image capture systems, etc over
11 serial and UDP. MAVLink packets will be routed to specific components when they
12 have a destination address. Any packet, targeted or broadcasted, can be
13 filtered based on source system/component, destination system/component and
14 message type. The filter can also apply a priority to packets allowing more
15 important packets to take priority over lower priority packets when an endpoint
16 is choked.

35 ## Install
36 
37 In order to compile you will need the following packages:
38 
39 * GCC 7+ or Clang 5+ (needed for C++17 support)
40 * [CMake v3.3+](https://cmake.org/)
41 * [Boost v1.54+](https://www.boost.org/)
42 
43 `mavtables` can be easily installed using the standard procedure of
44 ```
45 $ make
46 # make install
47 ```
48 The installation prefix is `/usr/local` by default but can be changed with
49 ```
50 $ make
51 # make PREFIX=/desired/install/path install
52 ```
